‚ÑπÔ∏è About SAM

Segment Anything Model (SAM) by Meta AI automatically segments objects in images.


Model Sizes:

  • ViT-B: Fast, good for real-time (85MB)
  • ViT-L: Balanced speed/quality (334MB)
  • ViT-H: Best quality, slower (2.4GB)

Backend Options:

  • PyTorch: Default, full compatibility
  • ONNX: Optimized inference, faster
  • TensorRT: GPU-optimized, fastest

Quality Filters:

  • Edge removal (border exclusion)
  • Size filtering (100-50K pixels)
  • Circularity threshold (>50%)
  • Stability score filtering
